# splice
splice is an app or app to be that helps people delegate tasks between each other.

Essentially a task distribtion app that allows people to split tasks between each other equally. You can make the tasks have different weightings so that tasks like washing the dishes and feeding the cat are worth different weights. 

# Ideas
- Each tasks has a weight
- Each user is assigned an equal weight of tasks per week
- Allow for a custom weighting distribution between the different users
- Tick box to show tasks complete per day
- Home screen widgets to see different tasks that are due
- All tasks are viewable on a shared list - called a home space
- Each user can see what tasks each other member in the home space has been assigned
